NXP DAL Documentation

Introduction

  • This document describes the NXP Driver Abstraction Layer (DAL) API definitions. DAL provides standard interface for the NXP NFC Reader Library and Application for the same. Hence, allowing easy portability of NXP NFC Reader Library on different Micro-controller SDKs and also Application to be independent of the underlying Micro-controller SDK being used.



  • Intended Audience:
    • This document is a reference to be used by the SW designers, implementers and integrators.

API Reference

  • Driver Abstraction Layer (DAL)
    Defines DAL APIs
  • Bus Abstraction Layer (BAL)
    Defines Bus Abstraction Layer API's
