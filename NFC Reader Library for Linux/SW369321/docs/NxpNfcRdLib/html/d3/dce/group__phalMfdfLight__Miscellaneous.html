Detailed Description

These Components implement the MIFARE DESFire Light Miscellaneous functions.

These are not part of actual DESFire Light data sheet rather its for internal purpose.

Returns
Status code
Return values

OtherDepending on implementation and underlying component.
Parameters
[in]pDataParamsPointer to this layer's parameter structure.
[in]wOptionDiversification option. 0xFFFF is for No Diversification.
[in]wKeyNoTMACKeyKey number in key store of Software.
[in]wKeyVerTMACKeyKey version in key store of Software.
[in]wRamKeyNoKey number of Destination Key where the computed session TMAC key will be stored. To be used for SAM AV3 only.
[in]wRamKeyVerKey version of Destination Key where the computed session TMAC key will be stored. To be used for SAM AV3 only.
[in]pDivInputDiversification input to diversify TMACKey.
[in]bDivInputLenLength of byte available in DivInput buffer.
[in]pTMC4 bytes Transaction MAC Counter. It should be 1 time subtracted from the actual value and shold be LSB first.
[in]pUidUID of the card.
[in]bUidLenLength of UID supplied.
[in]pTMITransaction MAC Input.
[in]wTMILenLength of bytes available in TMI buffer.
[out]pTMVThe computed Transaction MAC Value.
