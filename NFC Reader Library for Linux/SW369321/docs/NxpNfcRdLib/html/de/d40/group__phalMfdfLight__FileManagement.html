§ phalMfdfLight_CreateTransactionMacFile()

phStatus_t phalMfdfLight_CreateTransactionMacFile ( void *  pDataParams,
uint8_t  bFileNo,
uint8_t  bCommMode,
uint8_t *  pAccessRights,
uint8_t  bKeyType,
uint8_t *  bTMKey,
uint16_t  wKeyNo,
uint8_t  bTMKeyVer,
uint8_t *  pDivInput,
uint8_t  bDivInputLen 
)

Creates a Transaction MAC file.

An application can have only one Transaction MAC File.

Returns
Status code
Return values
PH_ERR_SUCCESSOperation successful.
OtherDepending on implementation and underlying component.
Parameters
[in]pDataParamsPointer to this layer's parameter structure.
[in]bFileNoFile number of the file to be created.
[in]bCommModeCommunication settings for the file.
  • PHAL_MFDFLIGHT_COMMUNICATION_PLAIN
  • PHAL_MFDFLIGHT_COMMUNICATION_PLAIN_1
  • PHAL_MFDFLIGHT_COMMUNICATION_MACD
  • PHAL_MFDFLIGHT_COMMUNICATION_ENC
[in]pAccessRightsThe access right to be applied for the file. Should be 2 byte.
[in]bKeyTypeBit0-1 indicates key type and it should be always PHAL_MFDFLIGHT_KEY_TYPE_AES128
[in]bTMKeyThe 16 byte key value to be used as TMKey.
[in]wKeyNoThe reference key number in SAM keystore. Only valid if initialized for SAM.
[in]bTMKeyVerThe version of the TMKey.
[in]pDivInputDiversification input to diversify the TMKey.
[in]bDivInputLenLength of bytes available in DivInput buffer.

§ phalMfdfLight_ChangeFileSettings()

phStatus_t phalMfdfLight_ChangeFileSettings ( void *  pDataParams,
uint8_t  bOption,
uint8_t  bFileNo,
uint8_t  bFileOption,
uint8_t *  pAccessRights,
uint8_t  bAdditionalInfoLen,
uint8_t *  bAdditionalInfo 
)

Changes the access parameters of an existing file.

Returns
Status code
Return values
PH_ERR_SUCCESSOperation successful.
OtherDepending on implementation and underlying component.
Parameters
[in]pDataParamsPointer to this layer's parameter structure.
[in]bOptionIndicates the mode of communication to be used while exchanging the data to PICC.
  • PHAL_MFDFLIGHT_COMMUNICATION_PLAIN
  • PHAL_MFDFLIGHT_COMMUNICATION_MACD
  • PHAL_MFDFLIGHT_COMMUNICATION_ENC Ored with the below option. Valid for Software mode only.
  • PHAL_MFDFLIGHT_MFM_SPECIFICS_ENABLED
[in]bFileNoFile number for which the setting need to be updated.
[in]bFileOptionNew communication settings for the file.
  • PHAL_MFDFLIGHT_FILE_OPTION_PLAIN
  • PHAL_MFDFLIGHT_FILE_OPTION_PLAIN_1
  • PHAL_MFDFLIGHT_FILE_OPTION_MACD
  • PHAL_MFDFLIGHT_FILE_OPTION_ENC
    Ored with one of the below flags if required.
  • PHAL_MFDFLIGHT_TMCLIMITCONFIG
  • PHAL_MFDFLIGHT_EXCLUNAUTHCONFIG
[in]pAccessRightsThe new access right to be applied for the file. Should be 2 byte.
[in]bAdditionalInfoLenLength of the Additional Info to be sent (2 Bytes or 4 Bytes). This is used to set TMC Limit in case of TMAC File
[in]bAdditionalInfoOne of the below inforamtion.
If Standard AES: 4 byte TMC Limit value.
If LRP : 2 byte TMC Limit value.
