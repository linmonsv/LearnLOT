ISO Select.

This command is implemented in compliance with ISO/IEC 7816-4.

Returns
Status code
Return values
PH_ERR_SUCCESSOperation successful.
OtherDepending on implementation and underlying component.
Parameters
[in]pDataParamsPointer to this layer's parameter structure.
[in]bOptionOption for return / no return of FCI.
  • PHAL_MFDFLIGHT_FCI_RETURNED
  • PHAL_MFDFLIGHT_FCI_NOT_RETURNED
[in]bSelectorThe selector to be used.
  • PHAL_MFDFLIGHT_SELECTOR_0
  • PHAL_MFDFLIGHT_SELECTOR_1
  • PHAL_MFDFLIGHT_SELECTOR_2
  • PHAL_MFDFLIGHT_SELECTOR_3
  • PHAL_MFDFLIGHT_SELECTOR_4
[in]pFidThe ISO File number to be selected.
[in]pDFnameThe ISO DFName to be selected. Valid only when bOption = 0x04.
[in]bDFnameLenLength of bytes available in DFname buffer.
[in]bExtendedLenApduFlag for Extended Length APDU. This should be always cleared.
[out]ppFCIThe FCI information returned by the PICC.
[out]pwFCILenLength of bytes available in FCI buffer.

)

ISO Read Binary.

This command is implemented in compliance with ISO/IEC 7816-4.

Returns
Status code
Return values
PH_ERR_SUCCESSOperation successful.
PH_ERR_SUCCESS_CHAININGoperation success with chaining.
OtherDepending on implementation and underlying component.
Parameters
[in]pDataParamsPointer to this layer's parameter structure.
[in]wOptionOne of the below options.
  • PH_EXCHANGE_DEFAULT : To exchange the initial command to the PICC.
  • PH_EXCHANGE_RXCHAINING: To exchange the chaining command to PICC in case PICC returns AF as the status.
[in]bOffsetThe offset from where the data should be read.
[in]bSfidShort ISO File Id. Bit 7 should be 1 to indicate Sfid is supplied. Else it is treated as MSB of 2Byte offset.
[in]dwBytesToReadNumber of bytes to read. If 0, then entire file to be read.
[in]bExtendedLenApduFlag for Extended Length APDU. This should be always cleared.
[out]ppRxBufferThe data retuned by the PICC.
[out]pBytesReadLength of bytes available in RxData buffer.
