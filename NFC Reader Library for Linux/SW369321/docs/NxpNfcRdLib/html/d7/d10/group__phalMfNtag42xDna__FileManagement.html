uint8_t *  pAccessRights,
uint8_t  bAdditionalInfoLen,

)

Changes the access parameters of an existing file.

Returns
Status code
Return values
PH_ERR_SUCCESSOperation successful.
OtherDepending on implementation and underlying component.
Parameters
[in]pDataParamsPointer to this layer's parameter structure.
[in]bCommModeIndicates the mode of communication to be used while exchanging the data to PICC.
  • PHAL_MFNTAG42XDNA_COMMUNICATION_PLAIN
  • PHAL_MFNTAG42XDNA_COMMUNICATION_MACD
  • PHAL_MFNTAG42XDNA_COMMUNICATION_ENC Ored with the below option
  • PHAL_MFNTAG42XDNA_SPECIFICS_ENABLED
[in]bFileNoFile number for which the setting need to be updated.
[in]bFileOptionNew communication settings for the file.
  • PHAL_MFNTAG42XDNA_FILE_OPTION_PLAIN
  • PHAL_MFNTAG42XDNA_FILE_OPTION_PLAIN_1
  • PHAL_MFNTAG42XDNA_FILE_OPTION_MACD
  • PHAL_MFNTAG42XDNA_FILE_OPTION_ENC
    Ored with one of the below flags if required.
  • PHAL_MFNTAG42XDNA_SDM_ENABLED
[in]pAccessRightsThe new access right to be applied for the file. Should be 2 byte.
[in]bAdditionalInfoLenLength of the Additional Info to be sent (2 Bytes or 4 Bytes). This is used to set TMC Limit in case of TMAC File
[in]bAdditionalInfoOne of the below inforamtion.
If Standard AES: 4 byte TMC Limit value.
If LRP : 2 byte TMC Limit value.

Parameters
[in]pDataParamsPointer to this layer's parameter structure.
[in]bOptionIndicates the mode of communication to be used while exchanging the data to PICC.
  • PHAL_MFNTAG42XDNA_COMMUNICATION_PLAIN
  • PHAL_MFNTAG42XDNA_COMMUNICATION_MACD
  • PHAL_MFNTAG42XDNA_COMMUNICATION_ENC
[in]bFileNoFile number for which the setting need to be updated.

  • PHAL_MFNTAG42XDNA_FILE_OPTION_PLAIN

  • PHAL_MFNTAG42XDNA_FILE_OPTION_MACD
  • PHAL_MFNTAG42XDNA_FILE_OPTION_ENC
    Ored with
  • PHAL_MFNTAG42XDNA_SDM_ENABLED
[in]pAccessRightsThe new access right to be applied for the file. Should be 2 byte.
[in]bSdmOptionsOne of the below values to be used. Can be ORed.
  • PHAL_MFNTAG42XDNA_VCUID_PRESENT
  • PHAL_MFNTAG42XDNA_RDCTR_PRESENT
  • PHAL_MFNTAG42XDNA_RDCTR_LIMIT_PRESENT
  • PHAL_MFNTAG42XDNA_SDM_ENC_FILE_DATA_PRESENT
  • PHAL_MFNTAG42XDNA_SDM_TT_STATUS_PRESENT
    Must be ored with the above values.
  • PHAL_MFNTAG42XDNA_SDM_ENCODING_MODE_ASCII
[in]pSdmAccessRightsThe SDM access rights to be applied. Should be 2 bytes.
[in]pVCUIDOffsetVCUID Offset information. Should be 3 bytes.
[in]pSDMReadCtrOffsetSDMReadCtr value (LSB first). Should be 3 bytes.
[in]pPICCDataOffsetMirror position (LSB first) for encrypted PICCData. Should be 3 bytes.
[in]pTTPermStatusOffsetTag Tamper Permanent Status Offset value. Should be 3 bytes.
[in]pSDMMACInputOffsetOffset in the file where the SDM MAC computation starts (LSB first). Should be 3 bytes.
[in]pSDMENCOffsetSDMENCFileData mirror position (LSB first). Should be 3 bytes.
[in]pSDMENCLenLength of the SDMENCFileData (LSB first). Should be 3 bytes.
[in]pSDMMACOffsetSDMMAC mirror position (LSB first). Should be 3 bytes.
[in]pSDMReadCtrLimitSDM Read Counter Limit value. Should be 3 bytes.
