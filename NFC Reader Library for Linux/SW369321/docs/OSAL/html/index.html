NXP OSAL Documentation

Introduction

  • This document describes the NXP Operating System Abstraction Layer (OSAL). OSAL provides standard interface for the NXP NFC Reader Library and Applications for the same a generic OS abstraction layer API's, allowing NXP NFC Reader Library and Applications to be independent of the underlying Operating System.



  • Intended Audience:
    • This document is a reference to be used by the SW designers, implementers and integrators.
